# Jithendra Babu G --- Developer Portfolio

Welcome to my personal developer portfolio. This website represents my
journey as a software developer and showcases the projects, skills, and
technologies I have worked with. I built this portfolio to present my
work in a clean, professional, and modern way so recruiters and
developers can easily understand my capabilities.

This portfolio focuses on delivering a smooth user experience, modern
design, and real-world project demonstrations. It highlights my interest
in building scalable backend systems, interactive user interfaces, and
complete full‑stack applications.

Clone the repository

Navigate to the folder

cd Portfolio

Install dependencies

npm install

Run development server

npm run dev
